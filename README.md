# Что такое метод в js

```
Метод JavaScript это свойство, содержащее определение функции. Методы это функции, хранящиеся как свойства объекта.

Метод — это блок кода, который выполняется только при его вызове. 
В метод можно передавать данные, известные как параметры. 
Методы используются для выполнения определенных действий, и они
также известны как функции.
```

# МЕТОД charAt()
```
Метод charAt() возвращает символ с заданным индексом (позицией) в строке.
Индекс первого символа равен 0, второго 1, ...
Индекс последнего символа равен длине строки - 1 .
```
# МЕТОД slice()
```
Метод slice(start[, end]) возвращает подстроку, начи
на позиции start и заканчивается позицией end-1 или до
конца строки если не указывать параметры end. Если
значение end меньше чем значение start, то метод вернёт пуст
строку.
```
# МЕТОД concat()
```
Метод concat() объединяет две или более строковые знач
ения и возвращает новую строку. При этом все аргументы
преобразуются к строкам прежде чем выполнить конкат
енацию.
```
# МЕТОД trim()
```
Метод trim() удаляет из строки левый и правый отступы 
пробелов и табуляций. Вернёт новую строку без отступов.
```
# МЕТОД substring()
```
Метод substring(start[, length]) возвращает подстроку, начи
на позиции start и заканчивается позицией length-1 или до
конца строки если не указывать параметры length. Если
значение length меньше чем значение start, то метод вернёт пуст
строку.
```
# МЕТОД includes()
```js
Метод includes(searchValue [, position ]) возвращает true, если поисковое
слово searchValue есть в текущей строке; false — в против
ном случае. Обязательный параметр searchValue может быть любой
тип данных. Например: "abc".includes("a") => true, "
abc".includes("d") => false. Также можно использовать опциональный
аргумент position для поиска в строке начиная со стро
ки с нужной позиции. Это полезно при работе с фра�
ментами HTML, где элементы могут иметь одни и те
же названия.
```
# МЕТОД indexOf()
```js
Метод indexOf(searchString [, fromIndex ] ) возвращает
индекс начала первого вхождения поискового 
слова searchString в текущей строке. Индексы 
считаются от 0 («ноль»). Если таких слов
не было найдено, то -1 возвращается.
Начало поиска по умолчанию равно 0.
```
# Метод replace()
```js
Метод replace(searchValue, newSubStr) заменяет все вх
ождения поискового слова searchValue в текущей строке
newSubStr и возвращает результат. Замена производит
ся только на первое вхождение.
```

# Метод replaceAll() 
```js
Метод replaceAll(searchValue, newSubStr) заменяет все в
хождения поискового слова searchValue в текущей строке
newSubStr и возвращает результат. Все вхождений заменятся.
```
# МЕТОД repeat()
```js
Метод repeat(count) создает новый объект String из этой
строки повторенного count раз. При этом каждый
элемент будет повторен count-1 раз. Строка «abc».
repeat(3) вернёт «aaabbbccc».
```

# Метод Math.round()
```js
Метод round(x) округляет число x до ближ
айшего целочисленного значения. Если число не является
целым, то округление выполняется с учётом д
робной части числа (по модулю). Например;
Math.round(-0.5); // -0
Math.round(497.286); // 497
Math.round(497.886); // 498
```
# Метод Math.ceil()
```js
Метод ceil(x) округляет число x до больш
ей или равной некоторому целому числу. Обрат
ная функция floor(). Например:
Math.ceil(497.286); // 498
Math.ceil(-0.5); // -0
```
# Метод Math.floor()
```js
Метод floor(x) округляет число x до меньше
й или равный некоторому целому числу. Обра
тая функция ceiling(). Например:
Math.floor(497.286); // 497
Math.floor(-0.5); // -1
```
# Метод Math.max()
```js
Метод max(a, b [, c,...]) возвращает наиболее
великое из аргументов a и b. Возможно указание
дополнительных параметров в качестве последних
аргументов метода. Например:
var num = [3, 2];
console.log('Max of'+ num +'is:'+ Math.max(...num));
// Max of [3,2] is: 3
```
# Метод Math.min()
```js
Метод min(a, b [, c,...]) возвращает наименее
великое из аргументов a и b. Возможно указа
ти дополнительных параметров в качестве послед
них аргументов метода. Например:
var num = [3, 2];
console.log('Min of'+ num +'is:'+ Math.min(...num));
// Min of [3,2] is: 2
```
# Метод Math.random()
```js
Возвращает случайное число от 0 до 1. При
заданном количестве вызовов этого метода
создается постоянный поток рандомизирован
ых значений. Смотрите также метод randomInt().
Например:
for (let i=0;i<10;++i){
    console.log("Random number:", Math.random());
    }
    // Random number:.1735444439535589
    // Random number:.227070308928227
    //...
```
# Метод Math.pow()
```js
Метод pow(base, exponent) возводит base в степень
exponent. Это эквивалентно оператору ** в
JavaScript. Например:
Math.pow(2, 3); // 8
Math.pow(.5,.5); //.25
```
# Math.sqrt()
```js
Метод sqrt(x) возвращает корень x по формуле
квадратного уравнения. Если значение x равняется
0 или меньше ноль, то будет выброшена ошибка.
Например:
Math.sqrt(64);    // 8
Math.sqrt(-1);   // NaN
```
# Math.abs()
```js
Метод abs(x) возвращает модуль числа x без зна-
чения его знака. Для целочисленных типов это
эквивалентно функции |x|. Например:
Math.abs(-1);     // 1
Math.abs(Infinity);// Infinity
```
# isNaN()
```js
Функция isNaN(value) проверяет наличие необра-
жимой ошибки при выполнении арифметическох
операций с value и возвращает true если она есть.
Других значений false. Вы можете использовать
isFinite(), чтобы проверить только те числа, которые
не являются бесконечностью.
Например:
isNaN('foo');      // true
isNaN(true);       // false
isNaN([1]);        // false
isNaN({ foo : 'bar' });// false
isNaN(undefined);  // false
isNaN();           // false
```
